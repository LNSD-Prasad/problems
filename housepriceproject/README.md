# 🏠 House Price Prediction using Linear Regression (Python)

This project predicts house prices based on:
- Area
- Number of Bedrooms
- Age of House
- Distance from City

It uses **Pandas** and **Scikit-learn LinearRegression**.

---

## 📂 Files in this project

- `housepred.py` → Main Python program
- `pricedata.csv` → Dataset used for training
- `README.md` → Project documentation
